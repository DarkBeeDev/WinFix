# WinFix
This tool *WinFix* is an automated batch script of SFC (System File Checker) Script.
===========================================
    DarkBee - SYSTEM REPAIR TOOL v2.0
===========================================
Created by: DarkBee
Project: WinFix 

[DESCRIPTION]
This tool automates the "System File Checker" (SFC) to find and 
repair corrupted Windows files. Use this if you have weird 
software crashes or OS errors.

[HOW TO USE]
1. EXTRACT the folder (Do not run from inside the zip).
2. Right-click 'Repair.bat'.
3. Select 'RUN AS ADMINISTRATOR'.
4. Wait for the scan to reach 100%.
5. Restart your PC if the scan found/fixed errors.

[WARNING]
- This tool fixes SOFTWARE. It cannot fix hardware issues 
  like thermal throttling, bad PSUs, or "desk-smashing" damage.
- Always run as Administrator or the scan will fail.
- Wait until it hits 100% then restart else it wont fix anything
- Runs on only x64 based architecture.
